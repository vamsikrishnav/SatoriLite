SatoriLite — UI Layout Explorations

Three TOC placement options + two overall layout variations. Open this file in a browser.

Option A: TOC as sidebar tab

Toggle between file tree and TOC in the same left sidebar. Compact, no extra panels.

Option B: TOC in right sidebar (outline panel)

Dedicated right panel for document outline. Always visible when toggled. Editor + Preview stay in the middle.

Option C: TOC as toolbar popover

Click a button → floating menu with headings. Minimal UI footprint. Dismisses after navigation.

Bonus: Reading mode layout

Full-width preview with a narrow outline on the right. For reading/reviewing docs rather than editing.
