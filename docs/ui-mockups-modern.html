SatoriLite — Modern Layout Explorations

Alternative layouts inspired by modern editors. Compare with the current SatoriLite layout and the options in ui-mockups.html.

1. Minimal Focus Mode Typora-inspired

No visible chrome until you hover or use shortcuts. The editor IS the preview — true WYSIWYG. Sidebar slides in from left on hover/shortcut. TOC appears on scroll as a floating mini-map on the right edge.

2. Notion / Block-Style Notion-inspired

Page-centric view. Breadcrumb navigation at top. Cover/icon area. TOC auto-generated at top of page. Blocks are the content units. Clean whitespace.

3. IDE Panels VS Code-inspired

Activity bar on the far left. Collapsible panels. Editor tabs. Integrated terminal / command palette feel. Outline as a panel section. Power-user oriented.

4. Focused Writing iA Writer-inspired

Maximum content focus. Single centered column. Typography-first. Current paragraph highlighted, surrounding text fades. TOC via keyboard shortcut (Cmd+Shift+O) as a quick-picker overlay.

5. Zen Split Clean split

Editor left, preview right. No chrome. No sidebar until summoned. Floating pill toolbar appears on hover at top center. Ultra-clean for distraction-free writing with live preview.
