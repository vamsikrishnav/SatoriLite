Getting Started

Welcome to SatoriLite — an offline-first markdown wiki.

Prerequisites

  • A Chromium-based browser (Chrome, Edge, Arc)
  • A folder with .md files
  • That's it. No server needed.

Quick Start

Step 1: Open the app in your browser.

Step 2: Click "Open Folder" and select your vault.

Step 3: Start writing.

Linking Notes

Type [ to trigger autocomplete. Select a file to insert a markdown link.

All files stay on your local disk — git-friendly, accessible by any tool, zero lock-in.

Architecture

Browser → File System Access API → Local .md files
